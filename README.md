# ComShop

Sistema de Gestión de Comercio Electrónico estructurado bajo el paradigma de programación funcional, desarrollado de manera nativa en el lenguaje Go.

##  Autore:
- Luis Lituma

## 🚀 Características del Sistema
- ✔ **Gestión de Productos**: Catálogo modular e inmutable.
- ✔ **Gestión de Clientes**: Registro parametrizado de consumidores.
- ✔ **Gestión de Pedidos**: Motor de cálculo exento de efectos colaterales.
- ✔ **Gestión de Inventario**: Alertas automatizadas de quiebre de stock.
- ✔ **Gestión de Pagos**: Historial transaccional clasificado por tipología.
- ✔ **Reportes Consolidados**: Métricas operativas directas de ventas.

## 🛠 Especificaciones Técnicas
- **Lenguaje Base**: Go (Golang)
- **Paradigma Core**: Programación Funcional
- **Almacenamiento Inicial**: Serialización de archivos JSON planos
